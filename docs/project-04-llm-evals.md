# LLM Prompt Playground + Evals Project

## Objective
The aim of this project is to develop an LLM Prompt Playground + Evals which will facilitate effective prompting and evaluation of large language models (LLMs).

## Phase
**Fase 4**

## Key Technologies
- **Python**: The primary programming language for the project.
- **LLM SDK**: Utilized for interfacing with large language models.
- **JSON Schema**: Employed for structured data validation and documentation.
- **pytest**: Used for testing the implementation and ensuring code quality.

## Learning Outcomes
1. Understanding how to create effective prompts for LLMs.
2. Developing evaluation mechanisms to measure the performance of LLM responses.
3. Gaining experience with the integration of LLM SDKs in Python.
4. Learning about JSON Schema for data validation and type definitions.
5. Enhancing skills in testing code with pytest.

## Challenges
- Designing prompts that yield accurate and relevant responses from LLMs.
- Evaluating the quality of model responses objectively.
- Handling diverse input formats and ensuring data integrity with JSON Schema.
- Writing comprehensive tests to cover edge cases in code implementation.

## Code Artifacts
- **Prompt templates**: Templates for constructing LLM prompts.
- **Evaluation scripts**: Scripts to evaluate responses based on predefined criteria.
- **Validation schemas**: JSON schemas for validating input and output data.
